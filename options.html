About CIpher

CIpher uses Chrome's built-in Prompt API with Gemini Nano to intelligently detect and blur personally identifiable information (PII) on most web pages. All processing happens locally, ensuring your privacy is never compromised.

PII Detection Filters

Choose which types of personally identifiable information to detect and blur.

Email Addresses
Blur email addresses like user@example.com
Phone Numbers
Blur phone numbers in various formats
Social Security Numbers
Blur SSNs like XXX-XX-XXXX
Credit Card Numbers
Blur credit and debit card numbers
Physical Addresses
Blur street addresses and locations
Passwords
Blur visible passwords and credentials
API Keys & Tokens
Blur API keys, access tokens, and secrets
